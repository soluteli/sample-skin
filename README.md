# sample-skin

## 安装
项目依赖 `parcel` 和 `http-server`, 请先安装（全局或局部）这两个模块，   
然后安装项目依赖
```
npm i
```

## 预览
### 换肤原理
启动服务
```
npm run howto
```
- 类名切换方式 demo: `./01-how-to-change-skin/class.html`
- 动态加载 css 的 demo: `./01-how-to-change-skin/dom.html`

### less 生成主题文件（类名切换方式）
该方式生成一个包含所有主题 css 样式文件，通过切换 body 类名实现换肤
```
npm run less-class
```
